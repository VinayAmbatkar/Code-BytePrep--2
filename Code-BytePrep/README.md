This project is a React application scaffolded with Vite. It includes multiple pages and functionalities as outlined below.

Pages Included
Authentication Page - Page for user login and registration.
Dashboard Page - Main page after authentication, featuring user insights and navigation.
All Test Page - Page listing all available tests.
Start Test Page - Page where users can begin a test.
MCQ Page - Multiple Choice Questions page for test-taking.
Programming Page - Page for programming-related tests or exercises.
Submission Sasses Page - Page for submitting test results or assignments.
Submitted Test Page - Page displaying submitted tests and their statuses.
Pricing Page - Page outlining the pricing plans for various services.
